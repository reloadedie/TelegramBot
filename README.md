# TelegramBot
this is my anti-spam bot...using BOT API Telegram and C#

Func's:
1. Block user in Black List
2. Block user's message (List Banned Words)
3. Logging any messages and other in console

C# .net 5.0 Console App

Nuget:
  Telegram.Bot v17.0.0
  Telegram.Bot.Extensions.Polling v1.0.2
